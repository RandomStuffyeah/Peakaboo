jStart32
========

jStart32 works as Win32-EXE wrapper to start JAVA Applications running on Windows 2000 or Windows XP. Both tools, the wrapper and the configuration tool are written in Delphi.

The source code is released under the GPL (http://www.gnu.org/copyleft/gpl.html).
